# ChartRecordingTools
This is a simply chart recording tool for uGUI. I with use for develop VR gesture.<br />

uGUI上で動作するシンプルなリアルタイムプロットツールです。VRにおけるジェスチャ開発のデバッグ用ツールとして作っています。
